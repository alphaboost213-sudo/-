# ImgUniq — Уникализатор изображений

Сервис который принимает ссылку на изображение и выдаёт статическую ссылку, по которой каждый раз отдаётся новая уникализированная версия фото.

## Как деплоить на Railway

### Способ 1 — через GitHub (рекомендую)

1. Создай новый репозиторий на GitHub
2. Залей туда все файлы из этой папки
3. Зайди на [railway.app](https://railway.app)
4. New Project → Deploy from GitHub repo
5. Выбери репозиторий → Deploy
6. Через 1-2 минуты сервис будет живой

### Способ 2 — Railway CLI

```bash
npm install -g @railway/cli
railway login
railway init
railway up
```

## Файлы

```
main.py          — основной сервер (Flask)
requirements.txt — зависимости Python
Procfile         — команда запуска для Railway
railway.toml     — конфиг Railway
```

## Как пользоваться

1. Открой URL сервиса (Railway выдаст после деплоя)
2. Вставь ссылку на скриншот (одну или несколько)
3. Нажми "Добавить" или "Создать ссылки"
4. Получи статическую ссылку вида `https://твой-сервис.railway.app/img/abc123`
5. По этой ссылке каждый раз будет отдаваться новая уникализированная версия

## Что делает уникализация

Каждый раз при открытии ссылки:
- Добавляется случайный шум (незаметный глазу)
- Микросдвиг яркости и контраста (±0.5-1.5%)
- Случайный лёгкий blur или sharpening
- Микрокроп на 0-2 пикселя с ресайзом обратно
- Случайное изменение насыщенности

Все изменения незаметны визуально, но делают файл технически уникальным.

## Важно

- Сервис хранит ссылки в памяти (при рестарте Railway — ссылки сбрасываются)
- Если нужно постоянное хранение — скажи, добавлю SQLite или Redis
- Лимит batch обработки — 50 ссылок за раз
